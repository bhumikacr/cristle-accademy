BR and HR tag

Break Tag

The <br> tag in the HTML code inserts a line break in the text.

Example:
<br>

Horizontal Rule Tag

  • The <hr> tag is used to create a horizontal rule on the web page.
    It is an empty tag.
  • The web browser starts a horizontal rule from a new line, and any text that follow this tag is also displayed in a new line.

Example :
<hr>
